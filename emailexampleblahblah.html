Greg Larson turned my attention today to something that is both cool and useful - a Chrome extension for use with Google Docs called Draftback. It's entirely coincidental that this tool, like last week's, can be used to help catch plagiarism.

Draftback lets you view the revision history of a Google Doc that's been shared with you as though it were being typed right in front of you. I tried it on a couple of files I'd typed up a while ago, and it was really cool to see my own thought process at work - but let's be honest, this tool can be useful for catching plagiarism, as it makes it really obvious when significant portions of a document are filled in via copy/paste. As a note, this doesn't PROVE that a student plagiarized, but it can help shine some light on potential areas of interest.

It's free, click the link below to try it. Feel free to call on me if you'd like assistance in using it.
